# hospeDATE – Sistema de Gestión de Reservas

Aplicación web diseñada para la gestión de reservas de habitaciones, permitiendo administrar usuarios, habitaciones y reservas de manera dinámica.  
Este proyecto fue desarrollado en equipo como práctica de lógica de programación, manejo de datos y control de versiones.

---

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- LocalStorage
- Git y GitHub

---

## Funcionalidades

- Crear, listar, editar y eliminar usuarios
- Gestión de habitaciones disponibles
- Registro y administración de reservas
- Validación de datos del usuario
- Interfaz responsiva y fácil de usar
